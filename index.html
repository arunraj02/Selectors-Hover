Combinator Selectors

Types of Combinator Selectors

1.)Descendent Selector:

The Descendant Selector selects all elements that are Descendants of a specified element.

/*Selects all p element inside a div/

2.)Child Selector:

The Child Selector selects all elements that are descendents of a specified element.

/*Selects all p element that are direct children of a div/

Symbol for Child Selector is >

3.)Adjacent Sibling Selector:

It selects an element that is immediately proceded by a specified element.

/*Selects the p element that is immediately proceded by an p./
